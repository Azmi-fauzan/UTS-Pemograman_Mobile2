- 2. Apa perbedaan antara diskon per item dan diskon total transaksi? berikan contoh penerapannya dalam aplikasi kasir
-Jawaban : Diskon Per item artinya tarif setiap satuan itemnya akan diterapkan sedangkan diskon per total transaksi artinya semua barang belanjaan akan dihitung terlebih dahulu baru dikenakan diskon setelah diketahui total belanjaan
           Contoh : ()Diskon Peritem ( Setiap item dikenakan diskon 10%)
                     Harga Barang:
                           A = 10.000, harga yang harus dibayarkan untuk item A = 9.000 
                           B = 20.000, harga yang harus dibayarkan untuk item B = 18.000
                           C = 30.000, harga yang harus dibayarkan untuk item A = 27.000
                     Total Biaya Yang Harus dikeluarkan = 9.000 + 18.000 + 27.000 = 54.000
                    ()Diskon total harga ( 10% )
                      Harga Barang:
                           A = 10.000 
                           B = 20.000
                           C = 30.000
                       Total Biaya Yang Harus Dikeluarkan adalah = 10.000 + 20.000 + 30.000 = 60.000 - 60.000 * 10% = 54.000
